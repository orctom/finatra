Announcing first milestone release of Finatra 2.0.0 — ~50x faster than v1.6.0!

posted by cacoco on 2015-04-30 14:05:13 -0700

We’re pleased to announce the first milestone release of Finatra 2.0.0!

You can download the release from Maven Central.

Features:

  • Significant performance improvements over v1.6.0.
  • Powerful feature and integration test support.
  • JSR-330 Dependency Injection using Google Guice.
  • Jackson based JSON parsing with additional support for required fields, default values, and custom validations.
  • Logback MDC integration with com.twitter.util.Local for contextual logging across futures.

See the Finatra github page for more information on getting started.

– The Finatra Team
